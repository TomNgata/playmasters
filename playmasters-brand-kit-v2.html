Typography Decisions — sourced from this post
Main Title → Libre Baskerville
"August 2023 – Singles Tournament" uses a serif book-style font. Elegance for official event naming.
Org/Event Name → Teko Bold Condensed
"PLAYMASTERS BOWLING" and bracket labels use bold condensed all-caps. Authority and legibility.
Border Treatment → Ornamental Indigo #3A3B8E
The decorative navy-indigo border frames the content. Use for certificates, formal tournament result cards.
Background → Pure White
Official comms posts use white background — high contrast, print-safe, social-media legible.

Brand Guidelines
Usage Rules
✓ Do
  • Use Anton for all hero/display headings
  • Use Libre Baskerville only for tournament/event titles
  • Use Strike Red (#E82030) as the primary action colour
  • Use the bowling ball gradient (#8B1FA2 → #D42080) for stat highlights
  • Keep tournament result posts on white background with indigo border
  • Use navy dark (#080B3A) for 3D canvas sections
  • Use "PLAYMASTERS BOWLING" in full on first reference in Teko
  • Mobile-first layout for Nairobi on-the-go access
✗ Don't
  • Never use orange or gold — not in the brand assets
  • Never use Inter or Roboto for display/hero headings
  • Never place the logo on a busy photographic background without overlay
  • Never separate "PLAY" and "MASTERS" with different font weights
  • Don't use the ball-pink gradient as a background — only as accent
  • Never expose raw ETL/scraper endpoints publicly
  • Don't use the red outline stroke independently of navy fills
  • Don't paygate any player stats or training data
